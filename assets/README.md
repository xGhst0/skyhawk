# Screenshots

The main README references the images below. Capture them from a running
instance (the dark theme shows best) and drop them in here with these exact
names.

| File | What to capture |
|------|-----------------|
| `workspace.png` | The investigation workspace on the **Findings** tab — a few findings with severity dots, ATT&CK pills and evidence. This is the hero image. |
| `advisor.png` | A finding's **⚡ Advice** modal open, showing the phased, copy-pasteable commands. |
| `ingest.png` | The **Ingest** tab after analyzing a Chainsaw export — the preview with new / already-in-case badges. |
| `report.png` | A frozen, signed **formal report** (the print view), banner visible. |
| `agents.png` | The **Agents** tab with an enrolled host and the Collect control. |
| `map.png` | The **Network map** with a few linked, compromised nodes. |

Recommended width ~1600px, PNG. Keep the same theme across all shots.
